# Git commit & push template

# 1) Initialize repository (if not already):
git init
git checkout -b main

# 2) Stage and commit files:
git add -A
git commit -m "chore: initial scaffold"

# 3) Add remote and push (example using SSH):
git remote add origin git@github.com:<your-username>/<your-repo>.git
git push -u origin main

# Alternative: create & push using GitHub CLI (recommended if you have `gh` installed):
# gh repo create <your-username>/<your-repo> --public --source=. --remote=origin --push

# Notes:
# - Replace <your-username>/<your-repo> with your GitHub account + repository name.
# - If you prefer HTTPS, use https://github.com/<your-username>/<your-repo>.git as the remote URL.
